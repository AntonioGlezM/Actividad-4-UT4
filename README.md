# Actividad-4-UT4º

# nombre del proyecto:
 Diseño de un pedido de restaurante.

 # Descripción breve (qué hace)

 Hemos creado un programa por el cuál un restaruramte recibe los pedidos con toda la información del mismo. En él se va acualizando el estado del pedido a medida que el restaurante avanza con el pedido realizado.

 # Cómo ejecutar el progrma.

El programa se ejecuta compilando primero el archivo con el comando javac Main.java y una vez compilado ejecutamos el programa con java Main.
